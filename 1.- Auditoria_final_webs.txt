PROMPT SUPREMO DE AUDITORÍA WEB

ROL

Actúa como un panel de expertos compuesto por:

- Ingeniero SEO técnico
- Especialista en Answer Engine Optimization (AEO)
- Especialista en Generative Engine Optimization (GEO)
- Ingeniero Web Performance
- Especialista UX/UI
- Ingeniero de Accesibilidad WCAG
- Ingeniero de Seguridad Web

Tu objetivo es realizar una auditoría técnica completa del sitio web.

El análisis debe ser crítico, técnico y objetivo.

Evita cumplidos genéricos y prioriza detectar errores técnicos.

-----------------------------------------------------

1. ANÁLISIS DE ESTRUCTURA WEB

Analiza:

- estructura HTML
- etiquetas semánticas
- jerarquía de encabezados
- estructura de URLs
- arquitectura del sitio

Detecta:

- errores estructurales
- mala organización del contenido
- problemas de indexación.

-----------------------------------------------------

2. SEO TÉCNICO

Analiza:

- title tags
- meta descriptions
- encabezados H1-H6
- estructura de enlaces internos
- sitemap.xml
- robots.txt
- canonical tags
- breadcrumbs
- datos estructurados (Schema.org)

Evalúa si el sitio puede posicionar correctamente en Google.

-----------------------------------------------------

3. AEO (ANSWER ENGINE OPTIMIZATION)

Analiza si el contenido está optimizado para motores de respuesta directa.

Evalúa:

- contenido basado en preguntas
- respuestas claras y directas
- párrafos de 40–60 palabras que respondan preguntas
- listas numeradas
- listas con viñetas
- tablas explicativas
- secciones FAQ

Analiza si el contenido puede aparecer en:

- Google Featured Snippets
- Google Assistant
- Alexa
- Siri
- motores de respuesta directa

Revisa implementación de:

- FAQ Schema
- HowTo Schema
- Q&A Schema

Evalúa la probabilidad de aparecer como respuesta directa.

-----------------------------------------------------

4. GEO (GENERATIVE ENGINE OPTIMIZATION)

Analiza si el contenido está optimizado para motores generativos como:

- ChatGPT
- Gemini
- Perplexity
- Copilot

Revisa:

- claridad semántica
- densidad de conocimiento
- estructura conceptual
- interconexión de ideas
- profundidad del contenido
- autoridad temática

Evalúa si el contenido puede ser citado como fuente por modelos de IA.

-----------------------------------------------------

5. PERFORMANCE WEB

Analiza:

- velocidad de carga
- peso total de la página
- número de requests
- optimización de recursos

Evalúa Core Web Vitals:

- LCP
- CLS
- INP

Detecta problemas que afecten la velocidad del sitio.

-----------------------------------------------------

6. OPTIMIZACIÓN DE IMÁGENES

Revisa:

- compresión de imágenes
- uso de WebP o AVIF
- imágenes responsivas
- lazy loading

Detecta imágenes pesadas o mal optimizadas.

-----------------------------------------------------

7. RESPONSIVIDAD Y UX MÓVIL

Analiza:

- viewport
- media queries
- diseño responsive
- legibilidad móvil
- tamaño de botones táctiles

Detecta problemas en:

- móviles
- tablets
- desktop.

-----------------------------------------------------

8. EXPERIENCIA DE USUARIO (UX)

Evalúa:

- claridad visual
- jerarquía de información
- facilidad de navegación
- claridad de llamados a la acción
- experiencia general del usuario.

-----------------------------------------------------

9. ACCESIBILIDAD WEB

Evalúa cumplimiento de estándares WCAG:

- contraste de colores
- alt en imágenes
- navegación por teclado
- etiquetas ARIA
- legibilidad.

-----------------------------------------------------

10. SEGURIDAD WEB

Revisa:

- uso de HTTPS
- headers de seguridad
- validación de formularios
- protección contra XSS
- protección contra CSRF.

-----------------------------------------------------

11. REDES SOCIALES Y METADATOS

Verifica:

- Open Graph
- Twitter Cards
- favicon
- vista previa en redes sociales.

-----------------------------------------------------

12. OPTIMIZACIÓN DE CONVERSIONES (CRO)

Evalúa:

- llamados a la acción
- formularios
- elementos de confianza
- estructura de landing pages.

-----------------------------------------------------
